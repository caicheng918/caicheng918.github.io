Linux下编程准备

发表于2020-11-11|更新于2021-02-28
|字数总计:353|阅读时长:1分钟|阅读量:

一、在Linux下进行编程(必须知道Linux的root密码，不一定是开机密码，忘记见注明)

打开Linux，在命令行里面操作
1.打开命令行获得root权限 :
su root
2.输入root密码
3.安装emacs编辑器(需要联网):
sudo yum install emacs
4.都安装完成后打开emacs即可编写程序，编写完后保存为后缀为.c的文件
5.命令行编译如:
gcc hello.c -o hi
./hi

二、在windows下外挂Xshell进行编程

先在Linux下设置如下
1.打开命令行获得root权限:
su root
2.开启远程控制ssh服务:
首先安装ssh(需要联网):
yum install openssh-server
启动ssh:
systemctl start sshd.service
systemctl enable sshd.service
3.得到ip地址
命令行:ifconfig -f
2.打开Xshell
新建会话，在服务里面的主机处设置为Linux的ip地址，连接。出现established提示成功。

总结:
学校机房的电脑已经设置了ssh服务，但是不会告诉我们root密码，所以只能用外挂Xshell来进行操作，这其实与Linux下进行操作差不多。但出于学习目的，还是有必要装虚拟机从而直接在Linux下进行编程，这就不存在root密码的问题了，自己想怎么设置就怎么设置。

注明:
1.修改root密码命令:
\#sudo su
\#passwd root

2.外挂Xshell后，C源文件最后必须要空一行。(很迷)
