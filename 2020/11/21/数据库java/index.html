Java数据库编程

发表于2020-11-21|更新于2021-02-17
|阅读量:

因为题目只要求了编写三个实体类和实现类对数据库三个表进行操作，三个表之间联系没有要求，所以三张表的代码其实很相似，写出一个表的代码稍作修改就能写出三个表的代码。因为太懒导致注释缺失。

需要注意的似乎只有三个表的每个字段名的类型，不能定义错误了。在编写接口和继承类时都要注意到。

PS:我依据公众号里的代码做了修改，代码有重复的部分，仅供参考，参考，参考。

题目要求

1.根据数据库表的设计，编写相应的实体类（User，Problem，Answer），分别对应数据库的三张表。

2.编写相应的数据库访问接口（UserDao，ProblemDao，AnswerDao）以及该接口的实现类，实现对数据库表的增、删、改、查操作。
